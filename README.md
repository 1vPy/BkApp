功能包括：
 1.最近热映、即将上映的电影查看  
 2.电影详情页  
 3.电影收藏  
 4.评论、点赞功能  
 ......



使用到
 1.Dagger2  
 2.RxJava2  
 3.Retrofit2  
 4.OkHttp3  
 5.ButterKnife  
 6.MVP设计模式  
 7.Android Material Design
